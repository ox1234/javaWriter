移位操作

Map
<<：将操作数的二进制码整体左移指定位数，左移之后的空格使用“0”补充
>>：将操作数的二进制码整体右移指定位数，右移之后的空使用“符号位”来补充
            若是正数使用“0”填充
            若是负数使用“1”填充
>>>：将操作数的二进制码整体右移指定位数，右移之后的空位使用“0”来补充

public class Hello{
    public static void main(String[] args){
        
        //移位操作
        /*
            2的二进制：
            000000000_00000000_00000000_00000010
            左移3位
            000000_00000000_00000000_00000010000
            右移3位
            000000000_00000000_00000000_00000010
        */
        System.out.println(2<<3);//16
        /*
            -16
            原码：100000_00000000_00000000_00000010000
            反码：111111_11111111_11111111_11111101111
            补码：111111_11111111_11111111_11111110000
            右移3位:111111111_11111111_11111111_11111110(-2的补码)
        */
        System.out.println(-16>>3);//-2
        System.out.println(-16>>>3);
    }
}



位运算符

Map

遵循规则
&：参与运算的两个数，若相应位数的值都为1，则该位结果是1，否则是0
|：参与运算的两个数相应位中只要有一个1，结果就是1
^：参与运算的两个数相应位相同则结果为0，否则为1
～：表示把每个二进制位的“1”换成“0”，把“0”换成“1”

boolean的true和false在底层使用一位的1和0表示
true：1
false：0

public class BitOperatorDemo{
    public static void main(String[] args){
        /*
            遵循规则
            &：参与运算的两个数，若相应位数的值都为1，则该位结果是1，否则是0
            |：参与运算的两个数相应位中只要有一个1，结果就是1
            ^：参与运算的两个数相应位相同则结果为0，否则为1
            ～：表示把每个二进制位的“1”换成“0”，把“0”换成“1”

            3的二进制:00000011
            5的二进制:00000101

                00000011
            &    00000101
            ----------------
                00000001

                00000011
            |    00000101
            ----------------
                00000111

                00000011
            ^    00000101
            ----------------
                00000110

                
            ~    00000101
            ----------------
                11111010

        */
            
            System.out.println(3&5);//1
            System.out.println(3|5);//7
            System.out.println(3^5);//6
            System.out.println(~5);//-6

    }
}

//逻辑运算符
public class LogiclOperatorDemo{
    public static void main(String[] args){
        /*
            与：boolean ret=boolean表达式  与。boolean表达式B
                两个boolean表达式都为true，结果才是true，其他情况ret为false
            &:位与
            &&：短路与

            区别：
            &：如果A表达式为false，也会判断B表达式，再返回false
            &&：如果A表达式为false，则不再判断B表达式，直接返回false

            效率：短路与效率最高

            或：boolean ret=boolean表达式  与。boolean表达式B
                两个boolean表达式都为flase，结果才是false，其他情况ret为true
            |:即使A表达式为true，也会执行B表达式，无论B表达式结果是什么都是true
            ||：短路或
                只要A表达式为true，立即返回true，不再执行B表达式

        */

        System.out.println(true&false);//false
        System.out.println(true&true);//true
        System.out.println(false&true);//flase
        System.out.println(false&false);//false
        System.out.println("---------------------");
        System.out.println(true&&false);//false
        System.out.println(true&&true);//true
        System.out.println(false&&true);//flase
        System.out.println(false&&false);//false
        System.out.println("---------------------");
        //System.out.println(false&(1/0==1));//有错误，证明(1/0==1)执行了
        //System.out.println(false&&(1/0==1));//没有错误，证明(1/0==1) 没有被执行
        System.out.println(true|false);//true
        System.out.println(true|true);//true
        System.out.println(false|true);//true
        System.out.println(false|false);//false
        System.out.println("---------------------");
        System.out.println(true||false);//true
        System.out.println(true||true);//true
        System.out.println(false||true);//true
        System.out.println(false||false);//false
        System.out.println("---------------------");
        System.out.println(true^false);//true
        System.out.println(true^true);//false
        System.out.println(false^true);//true
        System.out.println(false^false);//false
        System.out.println("---------------------");
        System.out.println(!true);//false
        System.out.println(!false);//true


    }
}

表达式类型的自动提升

Map
当一个算数表达式中包含多个基本数据类型（boolean除外）的值时，整个算术表达式的数据类型将在数据运算时自动提升
所有的byter，short，char类型被自动提升到int类型
整个表达式的最终结果类型被提升到表达式中类型的最高类型

注意：String和任意数据用+链接起来，结果都是String

public class TypeConvertDemo{
    public static void main(String[] args){
        /*
            byte的二进制：00000101
            short的二进制：00000000_00000101
            int的二进制：00000000_00000000_00000000_00000101
        */

        
        byte b1=12;

        short s1=b1;//小转大
        int i1=s1;
        long l1=i1;
        float f1=l1;//long-->float
        double d1=f1;

        System.out.println(d1);

        float f2=(float)d1;//double-->float
        
        double pi=3.14;
        int i2=(int)pi;//double-->int
        System.out.println(i2);
        
        //直接把double类型的常量，赋给int类型变量13
        int i3=(int)(3.14+1.7);
        System.out.println(i3);

        double d2=123+1.1F+99L+3.14;

        byte b4=25;
        //b4变量加上3之后，再给b4变量
        b4=3+b4;//编译报错，b4为byte类型
    }
}

变量的定义和基本使用

Map
变量的定义：表示储存空间，可用来存放某一类型的常量，没有固定值，并可以重复使用，也可以用来储存某种类型的未知数据。

变量的特点：
1、占据着内存中的某一块储存区域；
2、该区域由自己的名称（变量名）和类型（数据类型）；
3、可以被重复使用；
4、该区域和数据可以在统一类型范围内不断变化；

//变量操作
public class VarDemo
{
    public static void main(String[] args)
    {
        /*
            1、变量的定义格式：
                数据类型 变量名；
            如：    int age;
            2、给变量赋值（初始化）
                变量名 = 声明变量类型的常量值；
            3、在声明变量的时候，并做初始化
                数据类型 变量名=声明变量名的常量值
                int age=17;

            注意：变量必须初始化（赋值）之后才能使用，初始化才是真正的在内存分配空间
        */
        int age;//定义了一个int类型的变量，变量名为age

        age=17;//表示把17这个常量值，赋给age变量
        System.out.println(age);//打印变量，其实打印的是盖变量当前所赋予的值

        //同时定义多个变量
        int a,b,c;//不推荐
    }
}
