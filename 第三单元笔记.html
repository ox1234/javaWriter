控制循环结构——continue

Map
continue：
跳过当前的循环，进入下一次循环操作
当前循环的循环体不再执行

public class ContinueDemo{
    public static void main(String[] args){
    //从1输出到10    
        for(int i=1;i<=10;i++){
            if(i==4){
                continue;//跳过当前循环，进入下一次循环
            }
            System.out.println(i);
        }
        System.out.println("ending-------");
    //求出100~200以内不能被3整除的数字
        for(int i=100;i<=200;i++){
            if(i%3==0){
                continue;
            }
            System.out.println(i);
        }
        
    }
}

控制循环结构——break

Map
break：
终止当前所在循环,之后的语句无法执行，不能编写

public class BreakDemo{
    public static void main(String[] args){
    //从1输出到10    
        for(int i=1;i<=10;i++){
            //如果输出7，则终止循环
            if(i==7){
                break;//终止循环
            }
            System.out.println(i);
        }
        System.out.println("-------------");
    //求出100以内前5个3的倍数的数字
    //3,6,9,12,15,18,21
    //1、循环出100以内所有数字
    //2、判断哪些数字是3的倍数
    //3、如果是3的倍数，用计数器递增1
    //4、当计数器的值为5的时候没必要继续循环
        int result=0;//第0个3的倍数
        for(int i=1;i<=100;i++){
            if(i%3==0){
                System.out.println(i);
                result++;
            }
            if(result==5){
                break;
            }
        }
        System.out.println(result);
    }
}

嵌套循环

Map
循环解决的是：某一个操作重复执行
如果一个重复的操作需要做N次，，此时得使用嵌套循环，可以把内层循环看成一个整体

注意：
1、使用循环嵌套时，内层循环和外层循环的循环控制变量不能相同
2、循环嵌套结构的书写，最好采用“右缩进”格式，以体现循环层次的关系
3、尽量避免太多和太深的循环嵌套结构

for循环

Map
格式:
for(初始化语句;boolean表达式;循环后操作语句){
    循环体语句
}
初始化语句：表示对循环初始化，只在循环开始时执行一次
boolean表达式：表达式为false时，循环终止，为true，才执行循环体
循环后操作语句：循环每次迭代后会调用该语句，一般是递增和递减

一般的，在for循环值的循环迭代变量起名规则：i，j，k，m，n


//for循环
public class ForDemo{
    public static void main(String[] args){
        /*
            for(初始化语句;boolean表达式;循环之后的操作语句){
                循环体
            }
            执行顺序：
                1、初始化语句
                2、boolean表达式：
                    true：进入第3步
                    false：跳过循环
                3、执行循环体
                4、执行循环之后的操作语句
                5、进入第2步
            注意：初始化语句只在最开始的时候执行一次
        */
        //叫500声帅锅，打印500次
        for(int count=0;count<500;count++){
            System.out.println("帅哥"+count);
        }
        //从1数到100，打印从1～100
        for(int num=1;num<=100;num++){
            System.out.println(num);
        }
        //计算100以内正整数之和
        int result=0;
        for(int number=1;number<=100;number++){
            result=number+result;
        }
        System.out.println(result);
    }
}

if与switch的选择

if和switch都属于选择语句，也就是说功能室相似的
if：判断条件是boolean类型的
switch：判断条件是整数表达式==int类型的值
—————————————————————————————
发现switch只能适合对整数的值判断，如果是其他类型的判断，只能使用if语句
选择：如果是对整数表达式判断等于多少，首选使用switch
           其他情况，使用if语句

switch的使用注意

Map
注意：
1、其实if语句更强大，switch仅仅是对整数类型的判断，使用更简单
2、switch支持的类型是byte, short,char,int
3、default表示所有的case值都不正确，一般放在switch的最后，也不需要使用break语句

本质：switch仅仅只能支持int类型（byte, short char会自动提升为int类型）

Case只是用来为switch选择一个执行代码的入口，如果一旦进入某一个入口之后，后面的case实际上已经失去了判断的意义，也就是说代码在进入switch执行的时候会把入口case之后的case通通忽略，会一直往下执行，直到遇到break活着return。（穿透）

public class SwitchDemo{
    public static void main(String[] args){
        //switch支持char类型
        char ch='C';
        switch(ch){
            case 'A':System.out.println("A");break;
            case 'B':System.out.println("B");break;
            case 'C':System.out.println("C");break;
                }
        //判断是工作日还是休息日
             int weekday=1;
        switch(weekday){
            case 1:
            case 2:
            case 3:
            case 4:
            case 5:System.out.println("工作日");break;
            case 6:
            case 7:System.out.println("休息日");break;
            default:System.out.println("亲没有这一天");
        }
    }
}

而if-else不能返回结果，只能控制语句的结构

注意：
1、不能在if(boolean表达式)之后使用分号
2、不能直接使用else语句
